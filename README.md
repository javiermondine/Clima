# Weather App

A weather application using Visual Crossing Weather API with temperature unit toggle and theme switching.

## Features

- Search weather by city
- Temperature toggle (°C / °F)
- Dynamic theme based on weather conditions
- Optional GIF integration with Giphy

## Setup

Provide API keys via:
- URL parameters: `?key=YOUR_VC_KEY&giphy=YOUR_GIPHY_KEY`
- localStorage: `localStorage.setItem('vc_key', 'KEY')`

## Technologies

HTML, CSS, JavaScript (ES Modules), Visual Crossing API, Giphy API
